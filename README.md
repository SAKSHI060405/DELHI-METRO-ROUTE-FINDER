# 🚇 Delhi Metro Route Finder

A modern and responsive Delhi Metro Route Finder built using HTML, CSS, and JavaScript.

## 📌 Features

- Select Source and Destination Stations
- Find Metro Routes
- Display Route Details
- Responsive Design
- Glassmorphism UI
- Metro-Themed Interface
- Interactive User Experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎨 UI Highlights

- Metro-themed background
- Glassmorphism card design
- Responsive layout
- Smooth hover effects
- Modern typography

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/delhi-metro-route-finder.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
